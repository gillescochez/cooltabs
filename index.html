CoolTabs (jQuery plugin)

CoolTabs is a jQuery plugin that create a COOL animated tab system. Watch the tabs move underneath the label as the content scroll by. You can use the jQuery easing plugin with it.

Otherwise not much to it, just try out below and use the download links below to download it.
